join-host

Client joins a host by hostId.

{
  "type": "join-host",
  "hostId": "abc123"
}
{
  "type": "join-host",
  "hostId": "abc123",
  "clientId": "xyz456"
}

msg-to-host

Client sends a message to its host.

{
  "type": "msg-to-host",
  "payload": "message data"
}

msg-to-client

Host sends a message to a specific client.

{
  "type": "msg-to-client",
  "clientId": "xyz456",
  "payload": "message data"
}
